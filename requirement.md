Website ini bukan hanya portfolio biasa, tetapi menjadi representasi digital sebagai seorang Software Engineer yang mampu membangun produk dari tahap ide hingga production.

Target utama website ini adalah:

- Recruiter
- Tech Lead
- CTO
- Client
- Startup Founder

Website harus memberikan kesan:

- Premium
- Modern
- Fast
- Professional

- Clean
- Engineering First

Website harus terlihat seperti website engineer kelas dunia, bukan template portfolio biasa.

# 25. Overall Impression

Saat seseorang membuka website ini, dalam 10 detik pertama mereka harus berpikir:

"Ini bukan sekadar programmer yang bisa membuat website, tetapi seorang Software Engineer yang mampu membangun dan mengelola produk digital secara end-to-end."

Website harus mencerminkan kemampuan mulai dari:

- Analisis kebutuhan
- Perancangan arsitektur sistem
- Pengembangan backend
- Pengembangan frontend
- Pengembangan mobile
- Integrasi API
- Integrasi sistem eksternal
- Deployment server
- Optimasi performa
- Maintenance aplikasi

Setiap detail—mulai dari desain, animasi, performa, hingga isi—harus memperkuat kesan profesional, modern, dan berorientasi pada kualitas engineering.
